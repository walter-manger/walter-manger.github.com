C Quickly

I skimmed through most of the first 4 chapters of C How To Program and then chapter 5 presents functions. I thought that I would continue skimming through, but then I got to the section on Recursion and the classic introduction to it through implementing the factorial function. 

#include <stdlib>

long factorial(long);

int main() { 
  int i;
  for(i = 1; i < 10; i++)
    printf("%2d! = %1d\n", i, factorial(i));

  return 0;
}

long factorial(long number) {
  if(number <= 1) {
    return 1;
  }
  return (number * factorial(number - 1));
}

I have to admit that I feel like I understand Recursion enough to be dangerous. In college, Recursion was hammered into my head pretty quickly and I was able to reason about it with some mathematical techniques, but I can’t say that I had full command of the subject. I am glad that the book has promised to discuss recursion in depth for the next few chapters. I am looking forward to attepting some of the exercises in this chapter.
